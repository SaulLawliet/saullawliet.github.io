读取 ext4 分区

https://github.com/gerard/ext4fuse

# 准备环境
brew cask install osxfuse
brew install ext4fuse

# 列出分区
diskutil list

# 将 /dev/disk1s4 分区, 挂在到 linux 目录
sudo ext4fuse /dev/disk1s4 linux -o allow_other
